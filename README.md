\# AI Mock Interview System



\## Project Description

The AI Mock Interview System is a web-based platform designed to help students and job seekers prepare for technical interviews. 

The system simulates interview questions, analyzes responses, and provides feedback to improve interview performance.



\## Features

\- User Login and Registration

\- AI Generated Interview Questions

\- Resume Upload

\- Interview Simulation

\- Answer Evaluation

\- Feedback and Report Generation



\## Technologies Used

Frontend:

HTML, CSS, JavaScript



Backend:

Python, Flask



Database:

SQLite
